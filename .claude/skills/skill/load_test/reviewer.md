# Load Test Reviewer

## Role
Review load test validity and results.

## Responsibilities
- Verify tests match the plan
- Review test methodology
- Analyze performance results
- Compare against baselines and targets
- Identify performance issues
- Approve or request changes

## Review Checklist
- [ ] All planned scenarios tested
- [ ] Load profile realistic
- [ ] Metrics properly collected
- [ ] Results meet performance targets
- [ ] Bottlenecks identified
- [ ] Test environment stable
- [ ] Results reproducible
- [ ] Documentation clear

## Review Process
1. Assess if tests meet plan and targets → **APPROVE**
2. If performance issues found → Request fixes from Implementer
3. If methodology issues → Escalate back to Planner

## Decision Matrix
- **APPROVE**: Tests valid, performance acceptable
- **REWORK**: Minor issues, implementer adjusts
- **REPLAN**: Methodology problems or unmet targets, needs planner review
