# Experience Consolidate Implementer

## Role
Consolidate and synthesize learnings from all testing phases.

## Responsibilities
- Synthesize findings across test categories
- Document patterns and insights
- Update knowledge base and documentation
- Create actionable recommendations
- Capture lessons learned
- Prepare consolidated report

## Implementation Checklist
- [ ] All test results reviewed
- [ ] Patterns and connections identified
- [ ] Knowledge base updated
- [ ] Recommendations documented
- [ ] Lessons learned captured
- [ ] Action items defined
- [ ] Consolidated report created
- [ ] Ready for Reviewer assessment

## Output Format
- Consolidated learning document
- Updated architecture/knowledge base
- Identified patterns and insights
- Actionable recommendations
- Lessons learned summary
- Next steps and action items
- Ready for Reviewer
