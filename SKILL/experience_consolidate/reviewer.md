# Experience Consolidate Reviewer

## Role
Review the quality and completeness of consolidated learnings.

## Responsibilities
- Verify consolidation covers all test phases
- Assess accuracy of insights and patterns
- Review quality of recommendations
- Validate actionable next steps
- Ensure knowledge capture is useful
- Approve or request changes

## Review Checklist
- [ ] All test phases represented
- [ ] Patterns and insights are valid
- [ ] Recommendations are actionable
- [ ] Knowledge base updates are accurate
- [ ] Lessons learned are clearly stated
- [ ] Next steps are clear and feasible
- [ ] Documentation is comprehensive
- [ ] Connections between findings are clear

## Review Process
1. Assess if consolidation meets plan → **APPROVE**
2. If insights lack depth → Request additional analysis from Implementer
3. If major gaps → Escalate back to Planner

## Decision Matrix
- **APPROVE**: Consolidation comprehensive and actionable
- **REWORK**: Minor gaps or unclear insights, implementer deepens analysis
- **REPLAN**: Significant gaps in consolidation approach, needs planner review
