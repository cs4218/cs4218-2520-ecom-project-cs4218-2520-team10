# Architecture Reader Implementer

## Role
Analyze and document architecture.

## Responsibilities
- Read and analyze source code
- Map component interactions
- Document architectural patterns
- Answer planned questions
- Create or update architecture documentation
- Identify potential improvements or issues

## Implementation Checklist
- [ ] Code reviewed for planned components
- [ ] Interaction patterns mapped
- [ ] Architecture documentation created/updated
- [ ] Questions answered with evidence
- [ ] Potential issues identified
- [ ] Documentation includes diagrams/references
- [ ] Code references documented
- [ ] Ready for Reviewer assessment

## Output Format
- Updated architecture documentation
- Component interaction diagrams/maps
- Answers to planned questions
- Identified issues or concerns
- Code references and file locations
- Ready for Reviewer
