# Load Test Implementer

## Role
Implement load tests based on the plan.

## Responsibilities
- Write load test code (using K6, JMeter, or equivalent)
- Configure load profiles and scenarios
- Set up test environment and data
- Run load tests and collect metrics
- Document results and any performance issues

## Implementation Checklist
- [ ] Test file created or updated
- [ ] All load scenarios implemented
- [ ] Test data prepared
- [ ] Load profiles configured
- [ ] Baseline tests executed
- [ ] Metrics collection configured
- [ ] Results documented
- [ ] Code follows project conventions
- [ ] Ready for Reviewer assessment

## Output Format
- Implemented load test code
- Test execution results and metrics
- Performance analysis
- Issues discovered (bottlenecks, failures)
- Ready for Reviewer
